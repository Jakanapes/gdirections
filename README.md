# Gdirections

Wrapper for Google's new Direction API.  Based on https://github.com/dpickett/toll_booth

## Installation

Add this line to your application's Gemfile:

    gem 'gdirections'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gdirections
